# PR Checklist

## General
- [ ] Scope is limited to the requested task
- [ ] No unrelated refactors
- [ ] Documentation updated if behavior changed

## Frontend
- [ ] `pnpm build` passes
- [ ] i18n updated
- [ ] No broken imports
- [ ] No hardcoded secrets

## Backend
- [ ] App starts correctly
- [ ] Auth/roles respected
- [ ] Tests updated or added where relevant
- [ ] No secrets committed

## Review notes
- [ ] Risks documented
- [ ] Manual test steps included
